> **High-performance native macOS 4chan media downloader.**
> Download entire threads, catalogs, or boards with intelligent organization, fail-safe resume, and zero duplicates.

---

## Features

- **Bulk Downloading** ‚ÄĒ queue up to 20 threads or catalogs simultaneously
- **Smart Organization** ‚ÄĒ automatic folder structure with WEBM separation
- **Fail-Safe Resume** ‚ÄĒ automatically resumes interrupted downloads
- **Duplicate Prevention** ‚ÄĒ SHA-256 hashing prevents re-downloading files
- **Rate Limiting** ‚ÄĒ adaptive throttling with exponential backoff prevents IP bans
- **Download Verification** ‚ÄĒ MD5 checksum validation ensures file integrity
- **Connection Pooling** ‚ÄĒ 4√ó connection pooling for faster concurrent downloads
- **Real-Time Progress** ‚ÄĒ live bandwidth monitoring with ETA display
- **Apple Silicon Native** ‚ÄĒ arm64 build optimized for M-series Macs

## Installation

1. Download the latest `4Charm.dmg` from [Releases](https://github.com/RazorBackRoar/4Charm/releases)
2. Open the DMG and drag `4Charm.app` to `/Applications`
3. First launch ‚ÄĒ right-click the app ‚Üí **Open** to bypass Gatekeeper on the ad-hoc signed build

## Usage

1. **Add URLs** ‚ÄĒ paste any 4chan thread, catalog, or board URL
2. **Start** ‚ÄĒ click Download and watch the live progress log
3. **Enjoy** ‚ÄĒ files are saved to your chosen download folder with automatic organization

## Development

### Requirements

- Python 3.14
- macOS 12.0+
- [uv](https://github.com/astral-sh/uv)

### Setup

```bash
git clone https://github.com/RazorBackRoar/4Charm.git
cd 4Charm
uv sync
uv run python -m four_charm.main
```

### Build

```bash
razorbuild 4Charm
# Output: dist/4Charm.dmg
```

### Lint & Test

```bash
uv run ruff check .
uv run ty check src --python-version 3.14
uv run pytest tests/ -q
```
