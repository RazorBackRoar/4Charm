4Charm - Advanced 4chan Media Downloader

4Charm is a high-performance macOS application designed for bulk downloading media from 4chan threads and boards.

KEY FEATURES:
• Bulk Downloads - Download all media from threads, catalogs, or entire boards
• Smart Organization - Automatically organizes downloads by thread with WEBM subfolder support
• Resume Capability - Automatically resumes interrupted downloads
• Duplicate Detection - Prevents downloading the same file twice using hash verification
• Concurrent Downloads - Multi-threaded downloading for maximum speed
• Adaptive Rate Limiting - Intelligent throttling to respect 4chan's API limits
• Custom Download Location - Choose where to save your files
• Dark Mode Support - Respects macOS system appearance settings

SUPPORTED MEDIA:
Images: JPG, PNG, GIF, WEBP, BMP
Videos: WEBM, MP4, MOV, AVI, MKV
Other: PDF, TXT, ZIP, RAR

HOW TO USE:
1. Launch 4Charm from your Applications folder
2. Choose your download folder (first time only)
3. Paste 4chan URLs (threads, catalogs, or boards)
4. Click "Start Download"
5. Monitor progress in real-time

SYSTEM REQUIREMENTS:
• macOS 10.13 (High Sierra) or later
• Active internet connection
• Sufficient disk space for downloads
